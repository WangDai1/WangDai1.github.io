Python-Appium的安装

发表于 2019-12-25

需要用到的软件如下：

  • JDK

  • Android-SDK

  • Python

  • Appium：服务端(执行代码)和桌面版 (调试)

  • Node.js: Appium是使用Node.js实现

  • Appium-python-Client

安装JDK

https://www.oracle.com/technetwork/java/javase/downloads/index.html

下载安装，然后开始配置变量

添加 JAVA_HOME 和 CLASSPATH

JAVA_HOME : C:\Program Files\Java\jdk-12.0.2 自己jdk路径

CLASSPATH值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar; 最前面是有“.”这个符号的不能漏掉

编辑 Path

Path：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin

按快捷键win+r，在运行内输入cmd，并输入Java，回车，如果出来的是些用法中文，那说明这是正确的，在接下来输入javac，回车，如果出现些用法中文，说明是环境变量配好了。

下载SDK

https://www.androiddevtools.cn

  • 下载安卓sdk

  • 安装完成之后 会自动打开 SDK Manager

  • 根据手机或模拟器的安卓系统版本下载对应的API

  • 设置安卓环境变量

    添加 ANDROID_HOME

    ANDROID_HOME: C:\Users\11072\AppData\Local\Android\android-sdk

    编辑Path

    Path: 

    %ANDROID_HOME%\tools
%ANDROID_HOME%\platform-tools

    配置完环境后，可以在运行cmd后输入adb，若报错则将platform-tools下有关adb的三个文件 复制到 tools下

安装Python

https://www.python.org/downloads/

下载合适的版本并安装

打开cmd输入python,出现版本号说明安装好了。

安装node.js

https://nodejs.org/en/download/

下载安装，安装完成后，运行cmd，输入node –v查看版本号

安装appium

appium-desktop

https://github.com/appium/appium-desktop/releases

下载安装相应版本

appium-server

npm install -g appium

若下载不成功可用 淘宝npm镜像cnpmjs.org可替代官方版本。打开输入:

npm install -g appium –registry=https://registry.npm.taobao.org

appium –v 查看版本号

若出现 powershell报错“无法加载xxx.ps1，因为在此系统上禁止运行脚本”

  1. 设置权限

​ 以管理员权限运行 powershell， 在打开的powershell命令行输入

  1. 使用PowerShell ， 不要使用PowerShell x86

安装Appium-Python-Client

进入cmd 输入：pip install Appium-Python-Client 通过Python安装等待安装完成。

Frida install

发表于 2019-11-07

简介

frida 是一款基于 python+javascript 的 hook 框架，可运行在 android、ios、linux、win等各个平台，主要使用的动态二进制插桩技术

官方地址
github地址

插桩技术

是指将额外的代码注入程序中以收集运行时的信息，可分为源代码插桩 SCI 和二进制插桩 BI。

  • 源代码插桩 SCI

    Source Code Instrumentation，额外代码注入到程序源代码中。

  • 二进制插桩 BI

    Binary Instrumentation，额外代码注入到二进制可执行文件中。

    1. 静态二进制插桩 SBI，Static Binary Instrumentation，在程序执行前插入额外的代码和数据，生成一个永久改变的可执行文件。
    2. 动态二进制插桩 DBI
      ，Dynamic Binary Instrumentation，在程序运行时实时插入额外代码和数据，对可执行文件没有任何永久改变。

安装

一、PC端

注意安装版本，系列工具版本号要一致

查看 frida 版本号

二、Android 端

电脑 USB 连接已root安卓手机

1. 查看手机型号，下载系统对应版本的 frida-server

例如型号为 arm64-v8a，则下载 frida-server-12.7.5-android-arm64.xz

2. 将其解压缩生成 frida-server-12.7.5-android-arm64 文件

3. 将解压之后的文件push 到设备中，指定到 /data/local/tmp 路径下重命名为 frida-server

4. 运行Android 设备中的 frida-server

执行完毕后为运行状态。

保留此窗口 shell，以保证服务运行，关闭该shell 或者停止ctrl+c 则服务关闭。接下来的操作可另起shell 或该步骤命令另起 shell 执行。

5. 进行端口转发监听

27042 用于与frida-server通信的默认端口号,之后的每个端口对应每个注入的进程，检查27042端口可检测 Frida 是否存在。

6. 检查是否成功

执行frida-ps -U 命令成功输出进程列表，如下所示

执行frida -U -f com.xxx.xxx 进行连接，选择一个进程，等待一段时间则进入该应用

则在手机上打开Magisk Manager>设置,关闭Magisk下的Magisk Hide
