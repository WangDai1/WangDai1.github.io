Python + Appium 安装

需要用到的软件如下：

  • JDK

  • Android-SDK

  • Python

  • Appium：服务端(执行代码)和桌面版 (调试)

  • Node.js: Appium是使用Node.js实现

  • Appium-python-Client

安装JDK

https://www.oracle.com/technetwork/java/javase/downloads/index.html

下载安装，然后开始配置变量

添加 JAVA_HOME 和 CLASSPATH

JAVA_HOME : C:\Program Files\Java\jdk-12.0.2 自己jdk路径

CLASSPATH值：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar; 最前面是有“.”这个符号的不能漏掉

编辑 Path

Path：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin

按快捷键win+r，在运行内输入cmd，并输入Java，回车，如果出来的是些用法中文，那说明这是正确的，在接下来输入javac，回车，如果出现些用法中文，说明是环境变量配好了。

下载SDK

https://www.androiddevtools.cn

  • 下载安卓sdk

  • 安装完成之后 会自动打开 SDK Manager

  • 根据手机或模拟器的安卓系统版本下载对应的API

  • 设置安卓环境变量

    添加 ANDROID_HOME

    ANDROID_HOME: C:\Users\11072\AppData\Local\Android\android-sdk

    编辑Path

    Path: 

    %ANDROID_HOME%\tools
%ANDROID_HOME%\platform-tools

    配置完环境后，可以在运行cmd后输入adb，若报错则将platform-tools下有关adb的三个文件 复制到 tools下

安装Python

https://www.python.org/downloads/

下载合适的版本并安装

打开cmd输入python,出现版本号说明安装好了。

安装node.js

https://nodejs.org/en/download/

下载安装，安装完成后，运行cmd，输入node –v查看版本号

安装appium

appium-desktop

https://github.com/appium/appium-desktop/releases

下载安装相应版本

appium-server

npm install -g appium

若下载不成功可用 淘宝npm镜像cnpmjs.org可替代官方版本。打开输入:

npm install -g appium –registry=https://registry.npm.taobao.org

appium –v 查看版本号

若出现 powershell报错“无法加载xxx.ps1，因为在此系统上禁止运行脚本”

  1. 设置权限

​ 以管理员权限运行 powershell， 在打开的powershell命令行输入

  1. 使用PowerShell ， 不要使用PowerShell x86

安装Appium-Python-Client

进入cmd 输入：pip install Appium-Python-Client 通过Python安装等待安装完成。
